6️⃣ Especificación de Interacciones
Elemento Acción Resultado
+ Agregar libro Click Abre modal vacío con título "Agregar libro"
Botón Editar Click Abre modal con datos precargados, título "Editar libro"
Botón Eliminar Click Abre modal de confirmación con datos del libro
Campo de búsqueda Escribir Filtra en tiempo real por título, autor o ISBN
Select Categoría Cambiar Filtra libros por categoría seleccionada
Botones Paginación Click Navega entre páginas de 20 libros cada una
Guardar Formulario Submit Valida, guarda en localStorage, cierra modal
Cancelar Modal Click Cierra modal sin guardar cambios

8️⃣ Especificación Técnica

Almacenamiento:

localStorage Key: books_library

Formato: JSON Array con hasta ~100 libros

Estructura de Datos:

{
  "id": "1234567890",
  "title": "El Quijote",
  "author": "Miguel de Cervantes",
  "isbn": "978-0-06-093546-7",
  "quantity": 5,
  "price": 29.99,
  "category": "Ficción"
}

Paginación:

Items por página: 20

Cálculo: (currentPage - 1) * 20 = startIndex

Total páginas: Math.ceil(totalBooks / 20)
